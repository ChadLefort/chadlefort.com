## Problem

Separate carts for memberships, campus events, and future offerings meant duplicated logic, friction for users, and fragmented analytics. Conversion on the membership flow had been flat for months.

## Solution

I built a single, mobile-first cart that replaced every legacy checkout with a shared cart type system. The UI was rebuilt in Vue 3 + Nuxt + Pinia with TypeScript end to end, reusable checkout components, and promotional-code support. Forms were validated with VeeValidate and edge cases were covered by unit tests using Vue Testing Library (~80% meaningful coverage) plus MSW for offline-first API mocks.

## Impact

- Membership signups exceeded their goal by **25%** after launch.
- Consolidated three cart experiences into one reusable cart engine.
- Cut frontend defects in half during the first quarter after launch.
