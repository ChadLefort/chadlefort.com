## Problem

The existing platform surface was scattered across static pages. Dentists had to dig through menus to find the features most relevant to their practice.

## Solution

I designed and shipped a widget-based dashboard in Vue 3 + Nuxt + TypeScript, using Vue's composable API and fully typed GraphQL bindings generated with GraphQL Code Generator. Each widget represented an area of the platform, so I became intimately familiar with the entire product while building. Mock Service Worker powered the dev experience, and Vue Testing Library held coverage at ~80%.

## Impact

- Views, clicks, and engagement climbed **325% to 9,564%** across measured widgets in 90 days.
- Survived multiple rounds of UX reviews and A/B tests before a clean launch.
- Unblocked future work by standardizing the dashboard widget contract.
