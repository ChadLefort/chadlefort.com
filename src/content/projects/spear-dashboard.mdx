## Problem

Static pages scattered the platform surface. Dentists had to dig through menus to find the features most relevant to their practice.

## Solution

I designed and shipped a widget-based dashboard in Vue 3 + Nuxt + TypeScript, using Vue's composable API and fully typed GraphQL bindings generated with GraphQL Code Generator. Each widget represented an area of the platform, so I learned the entire product while building. Mock Service Worker powered reliable local development, and I held coverage at ~80% with Vue Testing Library.

## Impact

- Views, clicks, and engagement climbed **325% to 9,564%** across measured widgets in 90 days.
- Survived multiple rounds of UX reviews and A/B tests before a clean launch.
- Unblocked future work by standardizing the dashboard widget contract.
