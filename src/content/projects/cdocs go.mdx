## Problem

Dental practices needed a way to deliver continuing education without requiring travel to in-person events. The client wanted an interactive platform where dentists, dental assistants, and their entire practice could join live instructor-guided courses and participate in member discussions from anywhere.

## Solution

As part of a team of three developers, I helped build CDOCS Go, a live online education SaaS platform offering weekly interactive courses and member discussions. The application was built with Vue 3, Nuxt, TypeScript, and GraphQL, using Bootstrap for styling. This was my first professional experience with both Vue and GraphQL, and it was one of the client's earliest projects leveraging Vue's composable API.

The platform used fully typed GraphQL type definitions generated with GraphQL Code Generator, client-side mocks powered by Mock Service Worker for reliable development and testing, and comprehensive end-to-end test coverage with Cypress.

## Impact

- Delivered a live education platform that let dental practices participate in weekly courses without travel.
- Helped establish Vue 3 composable patterns and typed GraphQL workflows that the team carried into future projects.
- Built with Cypress E2E coverage and MSW mocks from the start, keeping regressions low during active development.
